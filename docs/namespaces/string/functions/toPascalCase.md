# Function: toPascalCase()

> **toPascalCase**(`str`): `string`

Convert string to PASCAL_CASE,
collapsing non-alphameric sequences with a single underscore

## Parameters

• **str**: `string`

## Returns

`string`

## Example

```ts
toPascalCase("My friend\n the     1st sharK") // -> MY_FRIEND_THE_1ST_SHARK
```
